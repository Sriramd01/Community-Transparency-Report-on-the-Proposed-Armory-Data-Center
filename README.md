# Community Transparency Report on the Proposed Armory Data Center

This repository contains the report and supporting materials prepared to help community members better understand the proposed Armory Data Center in St. Louis, Missouri.

The work is shared as an objective transparency effort. Its purpose is not to advocate for or against the proposed project, but to present relevant data in a clear and accessible format so that residents, stakeholders, and decision-makers can better understand the issues involved.

## Repository Contents

- `Community Transparency report on the proposed armory data center.docx`
  Final report document.
- `Armory Data center project tables and calculations.xlsx`
  Supporting workbook containing tables, calculations, and data used in the analysis.

## Project Scope

The report and supporting analysis currently cover:

- estimated electricity demand associated with the proposed facility
- residential electricity bill context in Ameren Missouri territory
- population context in St. Louis City
- comparison with existing local data centers
- proximity to nearby sensitive institutions
- scenario-based backup generator emissions
- noise context based on source-level estimates, local ordinance tables, and EPA guidance
- zoning and land use context
